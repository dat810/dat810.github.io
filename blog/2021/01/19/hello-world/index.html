An Introduction to my Blog Website, Hooray!

Posted on January 19, 2021

Today I'm finally introducing my blog website to the public. I have been learning basic front-end web development for 3 months and this is my result. Though I used some code snippets from external websites, I will cover them later so that I can create more exciting widgets and put it on my website.

Currently, it is just a static website with no server side, but those server-based widgets will eventually be implemented on my website soon after, so stay tuned!

Needless to say, since I have launched by blog site, my most exciting moments will be recorded here. This includes a wide range of content, such as my projects, tutorials, and even some trivial stuff I encounter on the web. I'm glad I can share my thoughts with you.

Oh, by the way, let me recommend a very useful website, W3Schools.com. It is used by many people who wants to learn web development and computer programming! As I said before, I learned most of the code snippets from there! You should really check it out!

Lastly, as my footer said, let's do better!
